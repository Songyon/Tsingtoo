如何使用GitHub？

1.注册GitHub账号

这很简单

设置用户名和密码

这是GitHub主页，右上角有“Sign in”（登录）和“Sign up”（注册）按钮，我们需要点击“Sign up”

点击“Sign up”后，我们能看见这个表单

第一栏是用户名，第二栏是电子邮箱地址，第三栏是密码。
这里需要注意以下几点：
用户名：每次在你输入用户名之后，系统都会检查你填入的用户名是否已经被注册了。如果被注册了，那么会提示 Username is already taken.此时你需要另换一个用户名进行注册，昵称只能出现字母、数字、短横线，不能使用下划线(_)，开头不能使用短横线(-).也不能使用中文。
邮箱：要填写合法邮箱，并且是未在 GitHub 注册过的邮箱，否则会弹出错误提示。
密码：注意密码至少是7个字符。
填好后，点击“Create account”等待系统验证，这时候你可能会进行人机验证。

GitHub的幺蛾子

接下来会选择账户类型，我们直接免费账户就行，即点击“Unlimmited public repositories for free.”。
点击Continue继续后，你会被要求填一份问卷，可以直接乱填。

验证邮箱

你会被要求验证邮箱。
GitHub 将向您发送一封含有链接的电子邮件。 单击该链接后，您将转到 GitHub 仪表板并看到确认横幅。

如果不验证邮箱，你会将无法使用GitHub做最基本的操作。
验证邮箱这一步也可以参照GitHub Docs。
好了，大功告成！现在快去登录吧！
你的主页将是这样：GitHub.com/你的用户名

仓库相关

仓库是什么？

GitHub的官方描述是：仓库包含项目的所有文件，并存储每个文件的修订记录，您可以在仓库中讨论并管理项目的工作。
简而言之，存东西的地方。

创建仓库

这里参考GitHub Docs的教程
在任何页面的右上角，使用“+”下拉菜单选择 New repository（新建仓库）

为仓库命名，只能英文。 例如 “hello-world”

（可选）添加仓库的说明

选择仓库可见性
这里选择公开所有人可见就行。

（可选）Initialize this repository with a README（使用自述文件初始化此仓库）

单击 Create repository（创建仓库）

恭喜，您的第一个仓库建好了，仓库主页便是默认分支。

为仓库上传文件

仓库可以用Git工具上传文件，也可以在网页上上传不超过100个文件。
关于Git工具的介绍，你可以去GitHub Docs上的教程学习GitHub Desktop。
在仓库主页（的一个分支，一般是默认分支）点击“Add file”（添加文件）的“Upload files”（上传文件）

将文件拖入此框或者点击“choose your files”（选择文件）以选择文件

Commit changes 的第一栏是操作名称，一般系统会自动为你打出，第二栏是说明。
点击“Commit changes”以确认上传
